# 📚 StudySprint – Pomodoro Study Planner

StudySprint is a productivity web app designed to help students stay focused and organized using the Pomodoro Technique. It combines a study planner with a focus timer to improve concentration and build consistent study habits.

## 🚀 Features

- ⏱️ Pomodoro timer (focus + break sessions)
- 📋 Study planning and task organization
- 🎯 Helps build consistent study habits
- ⚡ Real-time timer updates
- 🎨 Clean and distraction-free UI
- 📱 Responsive design

---

## 🧠 About the Project

StudySprint is built to solve a common student problem: lack of focus and poor time management.

By combining structured time intervals (Pomodoro technique) with a simple planner, the app helps users break down study sessions into manageable chunks and stay productive without burnout.

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript

---

## ⚙️ How It Works

1. Start a Pomodoro session (usually 25 minutes)
2. Focus on your study task
3. Take a short break after the session
4. Repeat the cycle to maintain productivity
